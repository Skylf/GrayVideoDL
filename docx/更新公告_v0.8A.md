# 更新公告 v0.8A

**日期**: 2026-06-20  
**版本**: v0.8A

## 新增功能

### 下载项分享功能

为下载列表中已完成的任务添加分享按钮，点击后使用系统 `Intent.ACTION_SEND` 弹出系统应用选择器，让用户选择微信、QQ、备忘录等任意支持接收文件的 App。

#### 实现细节

- **入口**: `DownloadAdapter` 的列表项布局中增加分享按钮，仅在任务状态为"已完成"时显示
- **交互**: 点击分享按钮 → 根据文件类型构建 Intent → 调用 `startActivity(Intent.createChooser(...))`
- **兼容性**: 使用 `FileProvider` 提供 `content://` URI，确保 Android 7.0+ 跨应用文件访问
- **文件类型支持**: 自动识别 mp4/mkv/webm/m4a/mp3/aac 格式的 MIME 类型

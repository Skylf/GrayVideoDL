# GrayVideoDL 更新公告 — v0.5A

---

## 新增功能

### 1. 新增 B站大会员登录认证
支持在设置页面通过 WebView 扫码或密码登录 B站，登录后自动保存 Cookie，解锁大会员专属高画质格式。

### 2. 新增视频下载功能
选中画质后点击下载，视频将保存到设备 `Download/GrayVideoDL/` 目录，文件管理器中直接可见。

### 3. 新增独立日志页面
调试日志从首页独立出来，拥有专属页面，支持一键复制和清空，排查问题更方便。

### 4. 新增设置页面
- 默认画质选择：自动 / 4K / 2K / 1080p / 720p / 480p / 360p
- 调试日志开关：控制日志页面的显示与隐藏
- B站登录状态查看与管理

### 5. 新增首页 UI 优化
- 解析结果卡片分栏显示「视频画质」和「音频」，不再混杂
- 自动合并音视频开关，合并模式仅显示视频格式
- 解析成功顶部弹出「✓ 解析成功」动画提示
- 画质不满足时内联横幅提示，不再弹窗打扰

### 6. 新增重复解析检测
重复解析同一链接时弹出确认框，避免误操作反复请求。

### 7. 新增自动格式选择
根据默认画质设置自动选中最匹配的格式，无需手动挑选。

---

## 修复 Bug

### 1. 修复 B站视频解析返回 HTTP 412 错误
模拟 Chrome 浏览器完整请求头，添加 B站专用提取器参数，B站视频解析恢复正常。

### 2. 修复 yt-dlp 网络 DNS 解析失败
补全 AndroidManifest.xml 中的网络权限声明，应用现在能正常发起网络请求。

### 3. 修复首页 UI 多项体验问题
- 格式列表不再视频音频混排
- 未知大小的文件不再显示「未知大小」，改为友好提示
- 日志区域现在可以正常滚动
- 复制日志按钮不再被文字撑破

## 其他优化

- 底部导航栏支持 4 个 Tab：首页 / 下载 / 日志 / 设置
- 下载任务自动记录，重启应用不丢失
- 增加函数注册表自动化工具，便于后续开发维护
- 代码注释全面汉化，降低维护门槛

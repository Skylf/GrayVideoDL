### 全链路价值闭环

```
用户输入URL → HomeFragment解析 → 选择格式/画质
    → DownloadFragment接收任务 → Python yt-dlp下载
    → 进度轮询实时反馈 → 下载完成 → MediaStore复制到公共目录
    → LogFragment查看日志 · SettingsFragment管理设置/登录
```

## v0.3A — Bug 修复 & 调试增强（当前最新）

**日期**: 2026-06-09 | **标签**: `v0.3A` | **Commit**: `316f1fa`

### Bug 修复

| # | Bug | 根因 | 修复 |
|---|-----|------|------|
| 1 | **B站登录后状态仍显示未登录** | B站 H5 登录流程中 SESSDATA **不通过 Set-Cookie** 设置，而是通过跨域重定向 URL 参数传递，`CookieManager.getCookie()` 始终返回空 | 新增 `extractCookiesFromUrl()` 从 URL 查询参数手动提取凭证 |
| 2 | **SettingsFragment 退出登录编译错误** | `CookieManager.removeAllCookies()` 的 `ValueCallback` lambda 参数不匹配 | 修正 lambda 签名 `() → {}` → `value → {}` |

## v0.1B — 初始版本（核心功能开发）

**日期**: 2026-06-09 | **标签**: `v0.1B` / `v0.2B` | **Commit**: `defec87`

### 新增功能

#### 1. 视频解析与下载（核心链路）

| 模块 | 文件 | 功能 |
|------|------|------|
| 视频解析 | `HomeFragment.java` | 输入 URL → 调用 Python 解析 → 展示可用格式列表 |
| 格式选择 | `HomeFragment.java` | 自动合并音视频、分栏显示、自动选择默认画质、格式锁定标记 |
| 下载执行 | `ytdlp_bridge.py` | `downloadVideo()` + `downloadVideoWithProgress()` 带进度回调 |
| 进度轮询 | `DownloadFragment.java` v2 | 每 500ms 读取进度文件，实时更新百分比/速度/ETA |
| 公共目录 | `DownloadFragment.java` | MediaStore API 复制到 `Download/GrayVideoDL/` |
| 暂停/继续 | `DownloadFragment.java` | 取消标志文件机制，Python 检测后自动停止 |
| 任务持久化 | `DownloadTask.java` | JSON 序列化到内部存储，重启后恢复 |

#### 2. B站大会员登录认证

| 文件 | 功能 |
|------|------|
| `BilibiliLoginFragment.java` | 全屏 WebView 登录（扫码/密码），自动检测登录状态 |
| `BilibiliLoginDialog.java` | 传统登录对话框 + Cookie 管理 |
| Cookie 流程 | 登录 → 提取 Cookie → Netscape 格式保存 → yt-dlp 使用 |
| 退出登录 | 清除 SharedPreferences + Cookie 文件 + WebView 缓存 |

#### 3. 下载管理（DownloadFragment v2 + DownloadAdapter）

- RecyclerView 列表展示所有下载任务
- 实时进度条 + 下载速度 + 已下载/总大小 + ETA 剩余时间
- 暂停/继续切换、删除任务（含文件）
- 打开文件夹按钮（系统文件管理器）
- 自动刷新（有活动任务时每秒更新）
- 空状态提示

#### 4. 设置页面

| 功能 | 说明 |
|------|------|
| 默认画质 | 自动/4K/2K/1080p/720p/480p/360p |
| 调试日志开关 | 控制底部导航栏日志 Tab 显隐 |
| B站登录状态 | 展示已登录/未登录，点击登录或退出 |

#### 5. 独立日志页面

| 文件 | 功能 |
|------|------|
| `LogFragment.java` | 独立的日志展示页 |
| `LogBuffer.java` | 日志缓冲区单例，跨 Fragment 共享 |

#### 6. 数据模型层

| 文件 | 说明 |
|------|------|
| `VideoInfo.java` | 视频信息模型 + 格式子类 + 分辨率转换 + 锁定检测 |
| `DownloadTask.java` | 下载任务模型 + JSON 序列化 + 状态管理 |
| `LogBuffer.java` | 日志缓冲区单例 |

#### 7. 首次运行初始化

| 文件 | 功能 |
|------|------|
| `GrayVideoDLApp.java` | 自定义 Application，首次运行触发环境检查 |
| `InitCheckHelper.java` | 目录检查 + Python 环境检测 + 结果展示 |

#### 8. Python 后端

| 函数 | 功能 |
|------|------|
| `testEnvironment()` | 检测 Python 和 yt-dlp 版本 |
| `extractVideoInfo()` | 解析视频信息（含 Cookie 支持） |
| `downloadVideo()` | 下载指定格式视频 |
| `downloadVideoWithProgress()` | 带进度回调下载 + 取消标志检测 |

#### 9. UI 组件

- 画质警告横幅（内联提示替代模态框）
- 成功提示动画浮层（渐显→停留5秒→渐隐）
- 重复解析检测
- 深色模式适配
- 4 Tab 底部导航（首页/下载/日志/设置）

#### 10. 效率工具

| 文件 | 功能 |
|------|------|
| `register_java_classes.py` | 自动扫描 Java 文件并生成 Excel 函数注册表 |

### Bug 修复

| # | Bug | 根因 | 修复 |
|---|-----|------|------|
| 1 | **B站 HTTP 412 Precondition Failed** | yt-dlp 默认请求头被 B站 反爬拦截 | 添加 Chrome 完整 HTTP 请求头 + B站 提取器参数 |

| 3 | **UI 交互体验问题** | 格式混排、按钮太小、缺少反馈 | 分栏、36dp 按钮、成功浮层动画 |

## v0.3A — Bug 修复 & 调试增强

### Bug 修复

#### B站登录后状态仍显示未登录
- **根因**: B站移动端 H5 登录流程中，SESSDATA **不通过 Set-Cookie 响应头设置**，而是通过跨域重定向 URL 查询参数传递，导致 `CookieManager.getCookie(".bilibili.com")` 始终返回空
- **修复**: 新增 `extractCookiesFromUrl()` 方法，在 `shouldOverrideUrlLoading()` 中拦截含有 `crossDomain` 和 `SESSDATA=` 的 URL，从查询参数中手动提取登录凭证并保存
- **验证**: 登录后自动跳转 `m.bilibili.com`，Cookie 正确保存，UI 显示"已登录"，功能可用 ✅

#### SettingsFragment 退出登录编译错误
- **根因**: `CookieManager.removeAllCookies()` 的 `ValueCallback` lambda 参数类型不匹配（`() → {}` 应为 `value → {}`）
- **修复**: 修正 lambda 签名

### 新增功能

#### 调试日志体系
- `BilibiliLoginFragment.java` — TAG=`BiliLogin`，全流程 Logcat 日志（Cookie 清除、页面加载、导航拦截、Cookie 检测、文件保存）
- `SettingsFragment.java` — TAG=`SettingsFrag`，登录状态切换、回调、退出登录日志

## 新增功能

### 1. B站大会员登录认证
- 新增 `BilibiliLoginFragment` 全屏 WebView 登录页，支持扫码登录和密码登录
- 新增 `BilibiliLoginDialog` 登录对话框（传统方式）
- 登录后自动提取 Cookie 并保存为 Netscape 格式文件，供 yt-dlp 使用
- 设置页面支持查看登录状态和退出登录
- 退出登录时彻底清除 Cookie 文件和 WebView 缓存

### 2. 视频下载功能
- 新增 `downloadVideo()` Python 函数，支持通过 yt-dlp 下载指定格式
- 下载完成后通过 MediaStore API 复制到公共 Download/GrayVideoDL/ 目录
- 下载任务持久化保存（JSON 格式到内部存储）
- 下载确认对话框（显示视频标题和画质信息）

### 3. 设置页面
- 默认画质选择：自动 / 4K / 2K / 1080p / 720p / 480p / 360p
- 调试日志开关：控制日志 Tab 的显隐
- B站登录状态显示和管理

### 4. 独立日志页面
- 新增 `LogFragment` 独立日志展示页面
- 支持复制日志和清空日志
- 日志 Tab 由调试日志开关控制显隐

### 5. 首页 UI 优化 v2
- 自动合并音视频开关，合并模式下仅显示视频格式
- 格式分栏显示（视频画质 / 音频两个独立区域）
- 画质警告横幅（内联提示替代模态框）
- 成功提示动画浮层（渐显 → 停留5秒 → 渐隐）
- 重复解析检测（相同 URL 弹出确认对话框）
- 自动格式选择（根据默认画质设置自动匹配）
- 格式锁定标记（需大会员的格式显示 "🔒 需大会员"）

### 6. 函数注册表自动化工具
- 新增 `register_java_classes.py` Python 脚本
- 自动扫描 Java 文件并生成 Excel 函数注册表
- 每个 Java 文件独立表格，支持追加更新
